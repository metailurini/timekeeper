# TimeKeeper

## How to use
### 1. Start
 - Default time is 25 minutes
 - Specify time: `<base url>?second=<time>`

   Exp:
   ```
   https://metailurini.github.io/timekeeper?second=1500
   https://metailurini.github.io/timekeeper?second=25*60
   ```

### 2. Pause
 - Close current tab

### 3. Continue
 - Reopen current tab
